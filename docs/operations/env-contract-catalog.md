# Runtime Env Key Catalog (BUG-INFRA-ENV-CONTRACT-GAP)

This is the authoritative non-secret catalog of runtime env keys referenced by source code.
It is validated by `npm run guard:env-template-contract`.

Scope:
- `apps/api/src` (`process.env`, `requireEnv`, `optionalEnv`)
- `apps/emr-gateway/src` (`process.env`)
- `apps/web/src` (`import.meta.env`)

Rules:
- Add new env keys here or in canonical templates before merging.
- Remove stale keys when no longer referenced in runtime code.
- Never put secret values in this file.
